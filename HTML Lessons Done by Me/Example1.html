When you display the file in the browser, you will find that the line breaks we had introduced(by hitting the ENTER key) are ignorned by the browser.
This is because HTML has a special Line Break tag.

When you want to display your paragraph aligning to left, to right or to center align attribute can be used

The align attribute can take one of the four values LEFT, RIGHT, CENTER, JUSTIFY.

Horizontal rule

There are four attributes of horizontal rule. COLOR attribute defines the thickness of the line.
WIDTH attribute defines the length or width of the line.
ALIGN attribute defines the alignment of the line.
NOSHADE attribute is used to shade the line and they have the 3D kind of effect.

Any line is aligned to the center even without specifying the align attribute. That is because center is a default attribute value for the hr tag.
